Description

"Project Arrhythmia" is a musical bullet-hell in which all things beautiful are deadly. Avoid the musically timed pulse of blocks and bars, and in the process you might just find out what this mysterious "Project Arrhythmia" is.

Features

  • Arcade ModeA fun way to play user made levels with your friends in up to 8 player Local Co-Op.
  • Level EditorA robust and easy to pick up level editor for user created content, built right into the game.
  • Steam WorkshopFull integration with Arcade Mode and the game's Level Editor to allow easy uploading and downloading of user made levels.
